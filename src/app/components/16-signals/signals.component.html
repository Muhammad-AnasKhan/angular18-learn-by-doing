Signals

Signals are a way to manage state in an Angular application. They are similar to reactive programming concepts like Observables, but they are more powerful and easier to use.

Key Points

  • Signals are functions that return a value.
  • Signals are reactive, meaning they can be observed and updated.
  • !! Signals still work if we turn off component change detection. Hence it doesn't need zone.js !!

  • but even if change detection is off, when signals gets new value it automatically updates the UI, hence ordinary variables also get updated.
