Builtin Pipes

Date pipe

original value - {{date}}
after Pipe - {{date | date}}
original value - {{date}}
after Pipe - {{date | date:'MMMM/dd/yyyy'}}

after Pipe - {{date | date:'MMMM-dd-yyyy hh:mm:ss'}}

Json pipe

original value - {{student}}
after Pipe - {{student | json}}

Async pipe

original value - {{currentTime}}
after Pipe - {{currentTime | async | date : 'hh:mm:ss '}}
